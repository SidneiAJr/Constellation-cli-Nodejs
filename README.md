# Constellation CLI | Node.js Edition

Um gerador de projetos completo que cria a estrutura backend e frontend para você em segundos. Suporte a múltiplas linguagens, frameworks e níveis de complexidade.

---

## ✨ Funcionalidades

- ✅ 5 backends disponíveis (JavaScript, TypeScript, Java, PHP, C#)
- ✅ 3 níveis de dependência (Básico, Avançado, Enterprise)
- ✅ Frontend de teste opcional (HTML + CSS + JS + API client)
- ✅ Estrutura MVC pronta
- ✅ Arquivos de configuração gerados automaticamente
- ✅ Interface CLI amigável com menus interativos

---

## 🚀 Backends suportados

| Linguagem | Framework | Níveis |
|-----------|-----------|--------|
| JavaScript | Express | Básico, Avançado, Enterprise |
| TypeScript | Express | Básico, Avançado, Enterprise |
| Java | Spring Boot | Enterprise |
| PHP | Slim | Padrão |
| C# | ASP.NET Core | Enterprise |

---

## 📦 Instalação

```bash
npm install -g constellation-cli-nodejs
```

## Estrutura:

```bash
seu-projeto/
├── Backend/
│   ├── app/
│   │   ├── controller/
│   │   ├── model/
│   │   ├── service/
│   │   ├── repository/
│   │   └── routes/
│   ├── .env
│   ├── .gitignore
│   └── package.json (ou pom.xml, composer.json, .csproj)
└── Frontend/ (opcional)
    ├── index.html
    ├── style.css
    ├── api.js
    └── app.js
```
