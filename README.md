# 🌟 Constellation CLI | Node.js Edition

Um gerador de projetos completo que cria a estrutura backend e frontend para você em segundos. Suporte a múltiplas linguagens, frameworks, arquiteturas e níveis de complexidade.

---

## ⚠️ Aviso

O código gerado é um esqueleto funcional. Cabe ao desenvolvedor implementar a lógica de negócio, testes e segurança adicional conforme necessário.

## ✨ Funcionalidades

- ✅ 5 linguagens (JavaScript, TypeScript, Java, PHP, C#)
- ✅ 4 arquiteturas (MVC, DDD, Clean Architecture, Hexagonal)
- ✅ 3 níveis de dependência (Básico, Avançado, Enterprise)
- ✅ Frontend de teste opcional (HTML + CSS + JS + API client)
- ✅ Estrutura de pastas e arquivos pronta
- ✅ Arquivos de configuração gerados automaticamente
- ✅ Interface CLI amigável com menus interativos

---

## 🚀 Suporte por linguagem

| Linguagem | Framework | Níveis | Arquiteturas |
|-----------|-----------|--------|--------------|
| JavaScript | Express | Básico, Avançado, Enterprise | MVC, DDD, Clean, Hexagonal |

| Java | Spring Boot | Enterprise (fallback MVC) | MVC, DDD |
| PHP | Slim | Padrão | MVC, DDD |
| C# | ASP.NET Core | Enterprise (fallback MVC) | MVC, DDD |
